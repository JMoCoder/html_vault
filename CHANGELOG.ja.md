## [Unreleased]

## [0.7.2] - 2026-06-04

### Added

- 期限付き token リンク、共有管理 API、公開用のサニタイズ済み共有ページ、
  共有取り消しを備えたバックエンド連携の公開ノート共有機能を追加しました。
- 公開共有の作成前に、実行可能 HTML、フォーム/埋め込み、危険な URL scheme、
  ローカル/プライベート参照、疑わしい secret をブロックする共有安全チェックを追加しました。
- ローカルの `toggleGroup('id')` 折りたたみ操作だけをプラットフォーム内蔵操作へ変換する
  限定的な安全共有変換を追加し、公開共有ページでは元スクリプトと外部フォント/リソースリンクを除去します。
- リーダーページの共有 UI、設定内の共有リンク管理を追加し、リーダー内の内部リンクコピー操作を削除しました。
- ワークスペースの Agent 接続状態をサイドバーのプロフィール popover へ移動し、
  その popover の大きなアバターからも画像をアップロードできるようにしました。
- パッケージ、アプリ、PWA、Demo のキャッシュバージョンを `0.7.2` に更新しました。

## [0.7.1] - 2026-06-04

### Fixed

- デプロイ済みワークスペースでカード上の Original リンクが静的な `content/...` パスを開き、SPA fallback によってアプリ画面へ置き換わる問題を修正しました。ログインセッション付きの raw item API を使うようになりました。
- セルフホスト HTTP/HTTPS デプロイで同一 origin の API を自動検出しつつ、GitHub Pages と静的 Demo はバックエンドなしのまま維持しました。
- Playwright 検証中は service worker をブロックし、API 連携モードと静的 Demo モードのカード Original リンクをカバーするテストを追加しました。
- パッケージ、アプリ、PWA、Demo のキャッシュバージョンを `0.7.1` に更新しました。

## [0.7.0] - 2026-06-04

  安定版として昇格しました。

## [0.2.0] - 2026-05-29

### Added

- 拡張可能なフロントエンド i18n 辞書。

- `localStorage` による言語設定保存。
- ダーク/ライトモード切り替え。
- AI プロバイダー設定、利用規約、プロジェクト情報、更新ドキュメントを含む設定ページ。

- AI ナレッジアシスタントのプレースホルダー。操作選択、プロンプト入力、将来のデータベース影響ジョブ前の二重確認を含む。
- ナレッジベースファイルに基づく複数ターン会話向けのグローバル AI サイドバープレースホルダー。
- 現在のビューからランダムな項目を開く魔法の杖アクション。
- 将来のアーカイブワークフロー向けのアーカイブ済みライブラリビュー。
- 項目カードとリーダーのお気に入り/アーカイブアイコン操作。
- ワークスペースのカード表示/横長リスト表示切り替え。
- ローカルバックアップ/復元、自動バックアップスナップショット、WebDAV 設定、JSON エクスポート用の独立したデータ設定グループ。
- ブラウザーインストールとアプリシェルキャッシュ用の PWA manifest と Service Worker。
- 現在の結果ビューからアーカイブ済み項目を除外するツールバーのアーカイブフィルター。

### Changed

- システム UI は翻訳キーから描画し、ユーザー作成のタイトル、コレクション、タグは変更しません。
- 言語選択はサイドバー下部に配置。
- API Key はブラウザー保存から明示的に除外し、静的モードでは非機密のモデル設定のみ保存。
- 設定ページのアイコンボタンを視覚的に中央揃え。
- AI プロバイダー設定をサイドバー管理とプロジェクト情報から視覚的に分離。
- 左側のインポート入口と右側の AI 作成入口を分離。
- Logo とサイドバーナビゲーションが、設定ページやリーダーから主ワークスペースへ戻るように統一。
- Manifest 項目に `archived` メタデータフラグを追加。
- カード表示と横長リスト表示でタイトル、概要、タグ、操作エリアを安定して確保し、長い概要は省略表示して機能部品を隠さないように変更。
