# Changelog

Languages: [English](CHANGELOG.md) | [中文](CHANGELOG.zh-CN.md) | [日本語](CHANGELOG.ja.md)

## [0.3.7] - 2026-05-29

### Changed

- 左サイドバーのリサイズハンドルが横スクロールバーを発生させないように変更。
- 複数選択フィルターのポップオーバー上部に「いずれか/すべて」のタグ一致コントロールを配置。
- カード操作ボタンが上部レイアウト行を占有しないようにし、コレクション、ソース、日付を左下エリアに固定。
- パッケージ、アプリ、PWA キャッシュのバージョンを `0.3.7` に更新。

## [0.3.6] - 2026-05-29

### Added

- 設定ページにユーザープロフィールとアカウント/セキュリティのプレースホルダーセクションを追加。

### Changed

- 上部のフィルターと並び替えコントロールを別グループに分離。
- カードのメタ情報を左下に「コレクション/ソース」と日付として表示し、お気に入り/アーカイブ操作は右上に配置。
- ソースラベルを生成/インポートのシステム文言に統一。
- 要確認ライブラリビューを削除し、ノートはソースに応じて生成またはインポート分類へ入るように変更。
- 設定ページタイトルをシンプルに「設定」とし、データ設定を AI 設定より前に配置。
- パッケージ、アプリ、PWA キャッシュのバージョンを `0.3.6` に更新。

## [0.3.5] - 2026-05-29

### Added

- タグの複数選択フィルターに「いずれか/すべて」の一致モードを追加。
- MCP と Docker の両方のタグを持つサンプルノートを追加し、組み合わせタグフィルターを検証しやすくしました。

### Changed

- ライブラリナビゲーションをクリックするとコレクション/タグの複数選択制限を解除し、そのライブラリ内の全コレクション/全タグを対象にします。
- 複数選択フィルターの「全選択」はコレクション/タグ制限をリセットします。
- パッケージ、アプリ、PWA キャッシュのバージョンを `0.3.5` に更新。

## [0.3.4] - 2026-05-29

### Added

- 結果フィルターグループに並び替えメニューを追加し、現在のビューを新しい順、古い順、タイトル A-Z、タイトル Z-A で並べ替え可能にしました。

### Changed

- パッケージ、アプリ、PWA キャッシュのバージョンを `0.3.4` に更新。

## [0.3.3] - 2026-05-29

### Added

- 左サイドバー幅をドラッグ調整し、ローカルに保存できるように追加。
- 上部ツールバーにコレクションとタグの複数選択フィルターポップオーバーを追加し、一括クリアに対応。
- AI コンテキスト表示にライブラリビュー、お気に入りのみ状態、アーカイブ表示状態、複数選択フィルター状態を追加。

### Changed

- 複数選択フィルターが有効な状態で左側のコレクションまたはタグをクリックすると、対応する複数選択グループと同期します。
- パッケージ、アプリ、PWA キャッシュのバージョンを `0.3.3` に更新。

## [0.3.2] - 2026-05-29

### Added

- 検索欄の右にグローバル AI アシスタントボタンを追加。
- 全ノート、コレクション、タグ、検索結果、リーダー状態に応じた文脈を表示する、幅調整可能な右 AI サイドバーを追加。
- 将来の Agent Server 向けに、AI チャットと HTML ノート生成のプレースホルダー操作を追加。

### Changed

- ナレッジベースに基づく複数ターン会話の計画を設定ページからグローバル AI サイドバーへ移動。
- パッケージ、アプリ、PWA キャッシュのバージョンを `0.3.2` に更新。

## [0.3.1] - 2026-05-29

### Changed

- 横長リスト表示で「読む/原文」操作をタグ行の下端に揃えるよう上へ移動し、行の高さをよりコンパクトに変更。
- パッケージ、アプリ、PWA キャッシュのバージョンを `0.3.1` に更新。

## [0.3.0] - 2026-05-29

### Added

- アーカイブフィルターの左に「お気に入りのみ」ツールバーフィルターを追加。
- アーカイブフィルターをデフォルトで有効化し、アーカイブ済み項目を初期表示から除外。

### Changed

- アーカイブフィルター有効時のツールチップを、クリック後の動作を示す「アーカイブ済みを表示」に変更。
- ツールバーフィルターの状態表示を枠ではなくアイコン色で表現。
- パッケージ、アプリ、PWA キャッシュのバージョンを `0.3.0` に更新。

## [0.1.0] - 2026-05-29

### Added

- 初期の静的優先 HTML Vault MVP。
- Manifest v2 ナレッジ項目モデル。
- サイドカー YAML メタデータ対応。
- Karakeep 風カードワークスペースとサイドバー絞り込み。
- iframe リーダーと原文を開く操作。
- 新規ナレッジ入口と任意 Agent API の形。
- サンプルコンテンツ、テスト、Docker 静的イメージ、README、MIT ライセンス。

## [0.2.0] - 2026-05-29

### Added

- 拡張可能なフロントエンド i18n 辞書。
- 中国語、英語、日本語のシステム UI 切り替え。
- `localStorage` による言語設定保存。
- ダーク/ライトモード切り替え。
- AI プロバイダー設定、利用規約、プロジェクト情報、更新ドキュメントを含む設定ページ。
- コレクション/タグ管理セクション。サイドバー表示制御と将来のメタデータ書き込み操作を用意。
- 固定システムビューの表示を制御するライブラリ管理セクション。
- AI ナレッジアシスタントのプレースホルダー。操作選択、プロンプト入力、将来のデータベース影響ジョブ前の二重確認を含む。
- ナレッジベースファイルに基づく複数ターン会話向けのグローバル AI サイドバープレースホルダー。
- 現在のビューからランダムな項目を開く魔法の杖アクション。
- 将来のアーカイブワークフロー向けのアーカイブ済みライブラリビュー。
- 項目カードとリーダーのお気に入り/アーカイブアイコン操作。
- ワークスペースのカード表示/横長リスト表示切り替え。
- ローカルバックアップ/復元、自動バックアップスナップショット、WebDAV 設定、JSON エクスポート用の独立したデータ設定グループ。
- ブラウザーインストールとアプリシェルキャッシュ用の PWA manifest と Service Worker。
- 現在の結果ビューからアーカイブ済み項目を除外するツールバーのアーカイブフィルター。

### Changed

- システム UI は翻訳キーから描画し、ユーザー作成のタイトル、コレクション、タグは変更しません。
- 言語選択はサイドバー下部に配置。
- API Key はブラウザー保存から明示的に除外し、静的モードでは非機密のモデル設定のみ保存。
- 設定ページのアイコンボタンを視覚的に中央揃え。
- AI プロバイダー設定をサイドバー管理とプロジェクト情報から視覚的に分離。
- 左側のインポート入口と右側の AI 作成入口を分離。
- Logo とサイドバーナビゲーションが、設定ページやリーダーから主ワークスペースへ戻るように統一。
- Manifest 項目に `archived` メタデータフラグを追加。
- カード表示と横長リスト表示でタイトル、概要、タグ、操作エリアを安定して確保し、長い概要は省略表示して機能部品を隠さないように変更。
