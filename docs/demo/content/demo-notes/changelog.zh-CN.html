[0.9.4] - 2026-06-10

变更

  • 优化知识库问答提示词，加入当前上下文摘要与近期对话上下文，使较短的追问也能继续锚定当前资料库上下文。
  • 优化多笔记上下文检索：根据近期对话拓展短问题检索词，并优先覆盖更多不同笔记，再补充同笔记的重复片段。
  • 新增检索覆盖诊断信息，包括上下文条目数、已覆盖条目数、覆盖率和查询是否拓展。
  • 包版本、应用版本、PWA、Demo 缓存版本和 provider user-agent 更新为 0.9.4。

修复

  • 修复 AI Markdown 渲染中有序列表全部显示为 1 的问题，并优化标题层级视觉效果；后端不限制模型固定回复格式。
  • 修复 AI 侧栏更多菜单的对齐和宽度。

[0.9.3] - 2026-06-09

修复

  • 优化可视化文件编辑器背景色控件：选中行内文字时，会跟随最近的非透明父级背景色。
  • 包版本、应用版本、PWA、Demo 缓存版本和 provider user-agent 更新为 0.9.3。

[0.9.2] - 2026-06-09

  • 非归档笔记新增保守的可视化文件编辑器，支持文本修改、文字/背景色、行内文字样式、撤销/重做、重置元素、可调宽/可收起编辑侧栏和预览区直接编辑。

变更

  • 优化左侧导航栏“资料库 / 集合 / 标签”标题的默认强调样式，提高可见性。
  • 包版本、应用版本、PWA、Demo 缓存版本和 provider user-agent 更新为 0.9.2。

修复

  • 修复可视化编辑器右侧文本框输入时光标跳到最前的问题。
  • 修复可视化编辑器点击空白区域会选中整页的问题，同时保留组合文本块整体选中能力。
  • 修复可视化编辑器文字颜色和背景色控件不同步的问题。
  • 修复可视化编辑器撤销/重做禁用状态和侧栏收起行为。

[0.9.1] - 2026-06-08

变更

  • 将 AI 侧栏中的“队列”统一调整为“生成历史”，并同步英文、中文、日文界面。
  • 生成历史按时间顺序展示，最新记录显示在弹窗底部。
  • 包版本、应用版本、PWA、Demo 缓存版本和 provider user-agent 更新为 0.9.1。

修复

  • 修复 AI 更多菜单中图标未居中、菜单项默认全部高亮的问题。
  • 修复阅读页 AI 会话历史未限定到当前笔记、错误显示全局会话的问题。
  • 优化生成历史行状态：已完成显示生成时间，生成中显示动态状态，失败且可重试的任务显示重试入口。

安全

  • 新增失败的“对话生成 HTML 笔记”任务重试接口，同时将重试元数据保留在后端私有任务记录中。
  • 上传资料生成任务默认不从历史中开放重试，避免为了重试而持久化用户上传资料正文。

[0.9.0] - 2026-06-08

新增

  • 新增服务端 AI 服务商配置，支持 OpenAI-compatible 接口，API key 只从后端环境变量读取。
  • 新增知识库问答 beta 流程，包含上下文解析、关键词检索、严格/内容拓展模式、Markdown 回复渲染、来源引用和限流。
  • 新增 AI 会话持久化、按上下文恢复最近会话、AI 侧栏内当前上下文历史，以及设置页全局会话管理。
  • 新增轻量 AI 任务队列、运行记录、队列弹窗，以及 job/run API。
  • 新增从已有 AI 对话生成 HTML 笔记的 beta 能力，采用 PM/UX/Coder/QA/Reviewer 分阶段图结构。
  • 新增从上传 HTML、Markdown 或纯文本资料生成 HTML 笔记的 beta 能力。
  • 新增 vector/hybrid 检索脚手架，在向量库未配置时自动回退到关键词检索。
  • 新增 AI 护栏，覆盖提示词预算、异常请求、疑似密钥输出和面向分享目标的 HTML 安全检查。
  • 优化 AI 面板前端：紧凑上下文头部、可调整输入区高度、回复状态、上下文重置、会话历史、队列/更多菜单行为，以及会话管理图标按钮。

变更

  • 更新英文、中文、日文 README，记录当前 AI 能力、服务端 AI 配置方式和 beta 限制。
  • 根据最新文档重新生成静态 Demo 的 README 与 CHANGELOG 示例笔记。
  • 包版本、应用版本、PWA、Demo 缓存版本和 provider user-agent 更新为 0.9.0。

安全

  • 确认 AI 服务商公开状态只返回 has_api_key 等公开字段；浏览器设置 API 不接受也不返回原始 API key。
  • 新增测试，确认 AI 服务商密钥不会通过状态、配置或生成运行记录泄漏。

修复

  • 优化全局上下文问答检索，使概览类问题可以使用完整选中上下文，而不是过度聚焦单篇笔记。
  • 修复静态构建复制逻辑，避免 app_static 被复制到自身内部。
  • 修复 AI 侧栏菜单状态、会话历史按上下文过滤、打开设置页时关闭右侧栏等细节，并补充 Playwright 覆盖。
