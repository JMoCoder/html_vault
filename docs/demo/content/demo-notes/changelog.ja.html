[0.9.2] - 2026-06-09

Added

  • アーカイブされていないノート向けに HTML ソース編集を追加しました。保存、保存して閉じる、全画面、折り返し、カーソル位置、保存前の共有安全チェックに対応します。
  • アーカイブされていないノート向けに保守的なビジュアルファイルエディターを追加しました。テキスト編集、文字色/背景色、インライン文字スタイル、undo/redo、要素リセット、リサイズ/折りたたみ可能な編集パネル、プレビュー内直接編集に対応します。

Changed

  • 左ナビゲーションの Library / Collections / Tags 見出しに、より分かりやすい既定のアクセント表示を追加しました。
  • パッケージ、アプリ、PWA、Demo cache、provider user-agent のバージョンを 0.9.2 に更新しました。

Fixed

  • ビジュアルエディター右側テキスト欄で入力中にカーソルが先頭へ戻る問題を修正しました。
  • ビジュアルエディターで空白キャンバスをクリックすると文書全体が選択される問題を修正しつつ、複合テキストブロックはまとめて選択できるようにしました。
  • ビジュアルエディターの文字色/背景色コントロールがレンダリング結果と同期しない問題を修正しました。
  • ビジュアルエディターの undo/redo 無効状態と、編集パネルの折りたたみ挙動を修正しました。

[0.9.1] - 2026-06-08

Changed

  • AI サイドバーの「Queue」を「生成履歴」に変更し、英語・中国語・日本語 UI を同期しました。
  • 生成履歴を時系列で表示し、最新の項目がポップオーバー下部に出るようにしました。
  • パッケージ、アプリ、PWA、Demo cache、provider user-agent のバージョンを 0.9.1 に更新しました。

Fixed

  • AI More メニューのアイコン位置と、項目が初期状態で強調表示される問題を修正しました。
  • リーダー画面の AI 会話履歴が全体履歴ではなく、現在のノートの履歴に絞り込まれるよう修正しました。
  • 生成履歴の行表示を改善し、完了済みジョブは生成時刻、実行中ジョブは動的ステータス、

失敗かつ再試行可能なジョブは再試行アクションを表示するようにしました。

Security

  • 失敗した conversation-to-HTML 生成ジョブの再試行 API を追加し、再試行メタデータは

バックエンドの非公開ジョブストアに保持するようにしました。

  • アップロード資料からの生成ジョブは、アップロード本文を永続化せず再試行できる場合を除き、

履歴からの再試行対象にしない設計を維持しました。
