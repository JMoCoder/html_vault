## [Unreleased]

## [0.7.3] - 2026-06-04

### 新增

- 公开分享页现在会保留安全的内联 CSS 和同页 `#anchor` 锚点链接，同时继续阻止外部
  CSS 资源和危险 CSS 特性。
- 针对 Chart.js/canvas 类笔记新增 `requires-static-export:chart` 分享安全反馈，明确这类
  笔记需要静态图表导出，而不是执行脚本。

### 修复

- 修复公开分享页错误嵌套完整 HTML 文档的问题：现在会把安全 head 样式放入分享页 head，
  并只把正文片段放入分享内容区。
- 增加导入反馈文字间距和卡片摘要区域余量，避免上传提示过近以及摘要/标签轻微贴边。
- 包版本、应用版本、PWA 与 Demo 缓存版本更新为 `0.7.3`。

## [0.7.2] - 2026-06-04

### 新增

- 新增基于后端的公开笔记分享能力，支持带有效期的 token 链接、分享管理 API、
  公开净化分享页和取消分享。
- 新增分享安全检查：公开分享前会阻止可执行 HTML、表单/嵌入内容、危险 URL
  协议、本地/私有引用和疑似密钥。
- 新增受限的安全分享转换：本地 `toggleGroup('id')` 折叠交互会转换为平台内置交互，
  同时公开分享页仍会移除原始脚本和外部字体/资源链接。
- 新增阅读页分享入口、设置页分享链接管理，并移除阅读页内部复制链接动作。
- 将工作台 Agent 连接状态移动到侧栏用户资料弹窗，并支持在该弹窗中点击大头像上传头像。
- 包版本、应用版本、PWA 与 Demo 缓存版本更新为 `0.7.2`。

## [0.7.1] - 2026-06-04

### 修复

- 修复部署环境中卡片层级“原文”链接错误走静态 `content/...` 路径的问题，现在会使用带登录会话的 raw item API，避免被 SPA fallback 路由拦截成工作台页面。
- 为自托管 HTTP/HTTPS 部署增加同源 API 自动识别，同时保持 GitHub Pages 和静态 Demo 不接入后端。
- Playwright 检查中屏蔽 service worker，并新增 API 模式与静态 Demo 模式下卡片“原文”链接的覆盖。
- 包版本、应用版本、PWA 与 Demo 缓存版本更新为 `0.7.1`。
