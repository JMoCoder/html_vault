# 更新日志

语言：[English](CHANGELOG.md) | [中文](CHANGELOG.zh-CN.md) | [日本語](CHANGELOG.ja.md)

## [0.4.14] - 2026-06-02

### 变更

- 将默认 Docker Compose 路径改为单容器自托管笔记本部署，同时提供前端和 `/api/*`。
- 将 Caddy Basic Auth 下沉为可选公网部署示例，不再作为默认启动依赖。
- 简化默认 Docker 环境变量，本地与局域网部署无需生成 `.env` 也能启动。

## [0.4.13] - 2026-06-02

### 新增

- 新增可复用自托管 Docker 部署资产：`Dockerfile.api`、`compose.prod.yml`、`deploy/Caddyfile` 和后端容器入口脚本。
- 新增前端运行时 `config.js` 支持，让部署站点可以同源访问 API，同时不把后端 token 写入浏览器代码。
- 新增 `.dockerignore`，并扩展自托管 Docker 启动、更新、备份、回滚部署文档。

### 变更

- 初始生产 compose 路径使用 `compose.prod.yml`；默认 compose 路径已在 `0.4.14` 简化。

## [0.4.12] - 2026-06-02

### 新增

- 新增 `GET /api/version`，返回后端版本和仓库元数据。
- “设置 > 关于项目”现在会显示运行版本，并检查 GitHub releases/tags 是否有可更新版本。
- 更新检查只显示提示，不会自动更新服务端。

## [0.4.11] - 2026-06-02

### 新增

- 新增轻量后端任务记录，持久化保存到 `meta/config/jobs.json`。
- 新增 `POST /api/rebuild` 与 `GET /api/rebuild/{job_id}`，用于显式触发重建并查询状态。
- 新增 `GET /api/uploads/{upload_id}`，上传 HTML 时也会返回 `job_id`，便于导入完成后继续查询。
- 新增完整核心 API smoke 测试，覆盖上传、列表查询、元信息编辑、搜索、内容读取、收藏、归档、取消归档和永久删除。
- localhost 前端现在默认使用 `http://127.0.0.1:8787` 作为后端 API，无需内联脚本配置即可进入本地笔记本模式。
- README 新增真实本地笔记本启动说明，覆盖 `data/content`、`data/meta`、API 服务和静态 Web 服务。
- 新增可复用部署安全基线文档，并加入后端 API Token 与 CORS 配置，为自托管生产环境做准备。

## [0.4.10] - 2026-06-02

### 修复

- 修复旧缓存 HTML 壳加载新版导入按钮脚本时，隐藏导入文件输入框尚不存在导致的启动崩溃。
- PWA Service Worker 对页面导航和 GET 静态资源改为优先请求网络，离线时再使用缓存兜底。
- 应用与 PWA 缓存版本更新到 `0.4.10`，促使浏览器请求修复后的页面壳与脚本，不再继续复用旧的 `0.4.9` 资源。

## [0.4.9] - 2026-06-02

### 变更

## [0.4.8] - 2026-06-01

## [0.2.0] - 2026-05-29

### 新增

- 可扩展前端 i18n 字典。
- 中文、英文、日语系统 UI 切换。
- 语言偏好保存到 `localStorage`。
- 暗色/亮色模式切换。
- 设置页：AI 服务商配置、用户协议、关于项目、更新文档。
- 集合/标签管理区，支持侧栏显示控制，并预留未来元数据写入动作。
- 资料库管理区，用于控制固定系统视图是否显示在侧栏。
- AI 知识库助理占位模块，包含任务类型、提示词输入，以及未来影响数据库前的二次确认逻辑。
- 全局 AI 侧栏占位模块，用于未来基于知识库文件的多轮对话。
- “手气不错”魔法棒入口，可随机打开当前视图中的条目。
- 资料库新增“已归档”视图，用于后续归档工作流。
- 条目卡片和阅读页新增收藏、归档图标操作。
- 主工作区支持方块卡片与横向条目视图切换。
- 设置页新增独立数据设置区：本地备份与恢复、自动备份快照、WebDAV 设置、JSON 导出。
- 新增 PWA manifest 和 Service Worker，用于浏览器安装和应用外壳缓存。
- 工具栏新增归档筛选按钮，可从当前结果视图排除已归档内容。

### 变更

- 系统 UI 文案由翻译 key 渲染；用户创建的标题、集合、标签保持原样。
- 语言选择位于侧栏底部。
- API Key 明确不进入浏览器存储；静态模式只保存非敏感模型偏好。
- 设置页图标按钮统一视觉居中。
- AI 服务商设置与侧栏管理、项目信息之间有视觉分隔。
- 左侧导入入口与右侧 AI 创建入口分离。
- 点击 Logo 和侧栏导航会从设置页或阅读页回到主工作区。
- Manifest 条目新增 `archived` 元数据字段。
- 方块卡片与横向条目视图为标题、摘要、标签、操作区预留稳定区域，长摘要以省略截断，不再挤掉功能模块。
