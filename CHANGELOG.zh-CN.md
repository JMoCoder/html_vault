# 更新日志

语言：[English](CHANGELOG.md) | [中文](CHANGELOG.zh-CN.md) | [日本語](CHANGELOG.ja.md)

## [0.3.17] - 2026-05-30

### 变更

- 左侧导航的滚动提示线与拖动高亮轨道统一，聚焦时不再挤动导航数字位置。
- 归档改为先确认、默认从集合/标签中隐藏，并且只在“已归档”栏目统计，同时保留原集合/标签/收藏元数据。
- 顶部导航的筛选/排序组仅保留收藏筛选，阅读页右上角改为“分享 + 直达 AI 面板”。
- 包版本、应用版本与 PWA 缓存版本更新为 `0.3.17`。

## [0.3.16] - 2026-05-29

- AI 创建区来源选择框改为内嵌自定义箭头，避免箭头贴近右边框。
- 包版本、应用版本与 PWA 缓存版本更新为 `0.3.16`。

## [0.2.0] - 2026-05-29

### 新增

- 可扩展前端 i18n 字典。
- 中文、英文、日语系统 UI 切换。
- 语言偏好保存到 `localStorage`。
- 暗色/亮色模式切换。
- 设置页：AI 服务商配置、用户协议、关于项目、更新文档。
- 集合/标签管理区，支持侧栏显示控制，并预留未来元数据写入动作。
- 资料库管理区，用于控制固定系统视图是否显示在侧栏。
- AI 知识库助理占位模块，包含任务类型、提示词输入，以及未来影响数据库前的二次确认逻辑。
- 全局 AI 侧栏占位模块，用于未来基于知识库文件的多轮对话。
- “手气不错”魔法棒入口，可随机打开当前视图中的条目。
- 资料库新增“已归档”视图，用于后续归档工作流。
- 条目卡片和阅读页新增收藏、归档图标操作。
- 主工作区支持方块卡片与横向条目视图切换。
- 设置页新增独立数据设置区：本地备份与恢复、自动备份快照、WebDAV 设置、JSON 导出。
- 新增 PWA manifest 和 Service Worker，用于浏览器安装和应用外壳缓存。
- 工具栏新增归档筛选按钮，可从当前结果视图排除已归档内容。

### 变更

- 系统 UI 文案由翻译 key 渲染；用户创建的标题、集合、标签保持原样。
- 语言选择位于侧栏底部。
- API Key 明确不进入浏览器存储；静态模式只保存非敏感模型偏好。
- 设置页图标按钮统一视觉居中。
- AI 服务商设置与侧栏管理、项目信息之间有视觉分隔。
- 左侧导入入口与右侧 AI 创建入口分离。
- 点击 Logo 和侧栏导航会从设置页或阅读页回到主工作区。
- Manifest 条目新增 `archived` 元数据字段。
- 方块卡片与横向条目视图为标题、摘要、标签、操作区预留稳定区域，长摘要以省略截断，不再挤掉功能模块。
