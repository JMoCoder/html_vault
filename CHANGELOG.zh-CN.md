# 更新日志

语言：[English](CHANGELOG.md) | [中文](CHANGELOG.zh-CN.md) | [日本語](CHANGELOG.ja.md)

## [0.1.0] - 2026-05-29

### 新增

- 初始静态优先 HTML Vault MVP。
- Manifest v2 知识条目模型。
- 旁车 YAML 元数据支持。
- 类 Karakeep 卡片工作台与侧栏筛选。
- iframe 阅读页、原文打开。
- 新知识入口与可选 Agent API 形状。
- 示例内容、测试、Docker 静态镜像、README、MIT 许可。

## [0.2.0] - 2026-05-29

### 新增

- 可扩展前端 i18n 字典。
- 中文、英文、日语系统 UI 切换。
- 语言偏好保存到 `localStorage`。
- 暗色/亮色模式切换。
- 设置页：AI 服务商配置、用户协议、关于项目、更新文档。
- 集合/标签管理区，支持侧栏显示控制，并预留未来元数据写入动作。

### 变更

- 系统 UI 文案由翻译 key 渲染；用户创建的标题、集合、标签保持原样。
- 语言选择位于侧栏底部。
- API Key 明确不进入浏览器存储；静态模式只保存非敏感模型偏好。
- 设置页图标按钮统一视觉居中。
- AI 服务商设置与侧栏管理、项目信息之间有视觉分隔。
- 左侧导入入口与右侧 AI 创建入口分离。
